# Feature Playbook

1. Read `.ai/rules/*.md`
2. Write a short plan using `.ai/prompts/plan.md`
3. Implement in the smallest vertical slice possible
4. Run `.ai/checklists/verify.md`
5. If backend is affected, verify CI and deploy implications

## Definition of done

- User-visible goal is complete
- Shared contracts are updated
- Verification evidence exists
